════════════════════════════════════════════════════════════════════════════════
  🎯 ¿QUÉ HACE ESTA APLICACIÓN?
════════════════════════════════════════════════════════════════════════════════

La aplicación replica TODA la funcionalidad del Excel original pero con:

   ✓ Interfaz moderna y atractiva
   ✓ Gráficos interactivos (barras, radiales, donas)
   ✓ Navegación intuitiva por pestañas
   ✓ Cálculos automáticos
   ✓ Clasificación de prioridades (Alta, Media, Baja)
   ✓ Guardar y cargar evaluaciones
   ✓ Exportar resultados a Excel
   ✓ Más rápido que Excel
   ✓ No requiere tener Excel instalado

════════════════════════════════════════════════════════════════════════════════
  📊 FUNCIONALIDADES PRINCIPALES
════════════════════════════════════════════════════════════════════════════════

1. Página de INICIO
   → Instructivo completo
   → Explicación de la escala de calificación
   → Barra de progreso de la evaluación

2. PROCESO ADMINISTRATIVO
   → Planeación (análisis contexto, plan estratégico)
   → Organización (estructura, procesos, PHVA)
   → Dirección (liderazgo, comunicación, cultura)
   → Control (sistemas, auditorías, acciones)

3. PROCESO OPERATIVO
   → Logística de Compras (entrada, inventarios)
   → Gestión de Producción (equipos, infraestructura, planeación, control)
   → Logística Externa (distribución, servicio al cliente)

4. RESULTADOS GENERALES
   → Cumplimiento general del establecimiento
   → Gráfico de barras por aspecto
   → Análisis radial multidimensional
   → Tablas detalladas

5. PRIORIDADES
   → Elementos de prioridad ALTA (< 60%)
   → Elementos de prioridad MEDIA (60-74%)
   → Elementos de prioridad BAJA (≥ 75%)
   → Gráfico de distribución

6. GUARDAR/CARGAR
   → Guardar evaluación actual (formato JSON)
   → Cargar evaluaciones anteriores
   → Exportar a Excel
