┌─ LESS IS MORE (Menos es más)

│  • Sombras más sutiles
│  • Gradientes más simples
│  • Bordes más finos
│
└─────────────────────────────────────────────────────────────────────────────

┌─ CONSISTENCY (Consistencia)
│
│  • Paleta de colores cohesiva
│  • Sistema de espaciado uniforme
│  • Bordes radius consistentes (8px, 12px, 16px)
│  • Sombras escaladas (sm, md, lg, xl)
│
└─────────────────────────────────────────────────────────────────────────────

┌─ HIERARCHY (Jerarquía)
│
│  • Colores más oscuros para contenido importante
│  • Gradientes para elementos interactivos
│  • Sombras para elevar elementos
│  • Tamaños de fuente diferenciados
│
└─────────────────────────────────────────────────────────────────────────────

┌─ ACCESSIBILITY (Accesibilidad)
│
│  • Contraste mínimo 4.5:1 para textos
│  • Colores no dependientes solo del color
│  • Estados hover claramente diferenciados
│  • Enfoque visible en elementos interactivos
