🆚 HTML vs STREAMLIT:

┌─ VERSIÓN HTML (app_gpp.html)
│
│  ✅ No requiere instalación
│  ✅ Funciona sin servidor
│  ✅ Fácil de compartir
│  ❌ Funcionalidad limitada
│  ❌ Sin persistencia de datos real
│
└─────────────────────────────────────────────────────────────────────────────

┌─ VERSIÓN STREAMLIT (app.py)
│
│  ✅ Funcionalidad completa
│  ✅ Persistencia de datos
│  ✅ Cálculos avanzados
│  ✅ Exportación a Excel
│  ❌ Requiere Python y Streamlit
│
└─────────────────────────────────────────────────────────────────────────────
